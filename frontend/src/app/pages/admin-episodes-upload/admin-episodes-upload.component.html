📂 Cargar Episodios desde CSV

📋 Formato esperado del archivo CSV

El archivo debe incluir las siguientes columnas en la primera fila (encabezado):

name air_date episode characters (opcional)
The Rickoning January 15, 2025 S05E01 https://rickandmortyapi.com/api/character/1,https://rickandmortyapi.com/api/character/2
Interdimensional Trouble January 22, 2025 S05E02
  • El código episode debe tener formato S##E## (ej. S05E01)
  • Si el episodio ya existe en la base de datos, se omite sin error
  • El campo characters acepta URLs separadas por coma
  • Tamaño máximo del archivo: 2 MB

⚠️ Nota importante:

Puedes eliminar todos los episodios cargados desde CSV usando el botón "🗑️ Borrar carga". Esta acción eliminará permanentemente todos los episodios locales y no se puede deshacer.
